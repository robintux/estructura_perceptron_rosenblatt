# Diagrama Interactivo del Perceptrón

Una visualización interactiva y educativa de la estructura del perceptrón, la forma más simple de red neuronal artificial.

## Descripción

Este proyecto presenta un diagrama interactivo que explica cada componente del perceptrón:

- **Unidades de Entrada**: Reciben las señales del entorno
- **Pesos Sinápticos**: Representan la fuerza de las conexiones
- **Sesgo (Bias)**: Parámetro que desplaza la función de activación
- **Función de Suma**: Calcula la combinación lineal de las entradas
- **Función de Activación**: Transforma la suma en la salida final
- **Salida**: El resultado de la clasificación o decisión

## Tecnologías

- React 18
- Vite
- Tailwind CSS
- Lucide React (iconos)

## Instalación

```bash
npm install
```

## Uso

```bash
npm run dev
```

La aplicación se abrirá automáticamente en tu navegador en `http://localhost:3000`.

## Build

```bash
npm run build
```

## Características

- Diagrama SVG interactivo
- Información detallada al pasar el cursor sobre cada componente
- Diseño responsive y moderno
- Animaciones suaves
- Totalmente educativo y accesible
